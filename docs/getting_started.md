## Install

```bash
# 将仓库下载至本地
git clone git@github.com:thoughts-lab/thoughts-lab.git

# 下载依赖
yarn install

# 构建开发环境版本
yarn run dev:mp-weixin
# 构建生产环境版本
yarn run build:mp-weixin
```

## Environment
